在OC中的`#pragma mark`写法如下：

```
#pragma mark - Actions
```

而在Swift中的写法为：

```
//MARK: - Actions
```

另外，`//TODO:`和`//FIXME:`既可以用在方法里面，也可以用在方法块的外面。
